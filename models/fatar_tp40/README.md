## Key Mapping and Definition
- In this project, I used a set of FATAR TP40 keys, famous for being found in Nord Stage and Kurzweil keyboards.

- In this keyboard, there was no need to invert diodes.

- All pin mapping and matrix details are in the PDF [schematics.pdf](schematics.pdf).
